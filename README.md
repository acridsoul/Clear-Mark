## How To Use ‚öíÔ∏è
- Find the Clear-Mark-Main Folder on your Computer
- Extract the folder Contents
- Open Chrome and navigate to `chrome://extensions/`
- Enable "Developer mode" using the toggle switch in the top right corner.
- Click "Load unpacked" and select the directory where you downloaded the Clear Mark Files.
- Click the extensions toolbar at the top, pin ClearMark
- Navigate to Documentary Area and select on a video to watch
- When the video loads click open the extension and click remove watermark
- Voila!! Enjoy
